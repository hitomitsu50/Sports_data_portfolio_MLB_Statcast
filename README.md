## 概要 (Overview)
2025年シーズンのMLB Statcastデータを活用し、カーブにおける「平均回転数 (release_spin_rate)」が、ボールの「技術的な変化量 (pfx_z)」および「物理的な総落差 (Total Drop)」にどのような相関があるかを定量的に分析したプロジェクトです。

## 分析の核心 (Core Analysis)
「回転数が多ければ多いほど、落差が大きくなる」という野球界の定説を検証するため、以下の2つの指標を軸に相関関係を可視化しました。

- **Technical Correlation**: 平均回転数と「技術的な垂直変化量（pfx_z）」の相関。
- **Physical Correlation**: 平均回転数と「物理的な総落差（Total Drop）」の相関。

## 技術的ポイント (Technical Highlights)
- **実戦データの厳選**: 統計的信頼性を確保するため、2025年シーズンにカーブを100球以上投じた投手のみを抽出。
- **データクレンジング**: 
    - 選手名を `First Last`（名 姓）形式に統一。
    - カンマ表示を排除したクリーンなデータ構造。
    - 小数点第3位（.3f）までの精密な計算。
- **可視化の最適化**: `Matplotlib` と `Seaborn` を用い、技術面（Blues）と物理面（Oranges）の対比を2画面構成で実装。

## 分析結果 (Results)

### 【相関分析：回転数 vs 各指標】
| 指標の組み合わせ | 相関係数 (Correlation) | 統計的評価 |
| :--- | :--- | :--- |
| **平均回転数 vs pfx_z (技術面)** | **0.208** | 弱い正の相関 |
| **平均回転数 vs Total Drop (物理面)** | **0.028** | ほぼ無相関 |

## 考察 (Insights)
分析の結果、回転数と落差の間には予想に反して極めて低い相関しか認められませんでした。

- **Technical Side (pfx_z)**: 相関係数0.208という数値は、回転数そのものよりも、回転軸（Spin Axis）や回転効率（Spin Efficiency）が変化量に寄与している可能性が高いことを示唆しています。
- **Physical Side (Total Drop)**: 相関係数0.028という「ほぼ無関係」な結果は、物理的な総落差が、リリースポイントの高さや球速、重力加速度といった要素に支配されており、回転数による影響は極めて限定的であることを裏付けています。

**結論として、現代MLBにおける「エグいカーブ」の正体は、単純な回転数の多寡ではなく、リリース高や球速を含めた複合的な物理パラメータの組み合わせであると推察されます。**
