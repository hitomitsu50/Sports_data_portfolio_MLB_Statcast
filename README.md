    - `Seaborn`の`barplot`を用いた、直感的に伝わるランキンググラフの作成
- **振り返り:** 以前作成したコードを改めて一から写経することで、データの結合（`concat`）や、フラグ立ての処理がよりスムーズに理解できるようになりました。
