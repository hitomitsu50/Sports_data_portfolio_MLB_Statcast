### 📊 次の分析テーマ (予定)

現在の結合済みデータフレーム (`df`) を使用し、以下の分析と可視化を順次行います。

1.  **Whiff% TOP10 投手の棒グラフ作成**: 空振り率の算出とランキング表示。
2.  ERA（防御率）と K%（奪三振率）の相関分析（散布図）。
3.  球種別（フォーシーム、スライダーなど）の回転数と空振り率の比較。
