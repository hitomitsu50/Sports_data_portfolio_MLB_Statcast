### 【2日目】2025年12月29日：打球速度（Exit Velocity）ランキング分析
本日の学習テーマは、強打者の証である「打球速度」の可視化と分析です。

- **YouTube動画:** https://www.youtube.com/watch?v=pWt6LtVenKU

- **学習内容:** -

データ抽出の最適化: launch_speed（打球速度）に特化したデータクリーニング（欠損値の除外）。

打者視点での集計: 昨日の投手視点とは異なり、batter（打者ID）をキーとしたグループ化処理。

正規表現・文字列操作: des（打席結果のテキスト）から、正規表現や分割処理を用いて打者のフルネームを抽出するロジックの実装。

発展的な可視化:

棒グラフの先端に数値ラベル（mph）を表示し、一目でデータが読み取れるデザインに変更。

指標の性質に合わせ、暖色系パレット（flare）を採用。

- **振り返り:**

トラブル解決の学び: 当初、投手名が表示されたりラベルが空白になったりする問題が発生しましたが、インデックスの整理とデータのフィルタリングを見直すことで解決できました。

「Be Water」の実践: 「名前をフルネームにしたい」「単位を明記したい」という直感的な改善案を即座にコードに反映させる過程で、ライブラリの仕様への理解が深まりました。
